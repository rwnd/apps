# RWND Apps

Public App Store support pages and legal documents for RWND apps.

## Trivia Gym

| Field | URL (after GitHub Pages is enabled) |
|---|---|
| **Support URL** | https://rwnd.github.io/apps/app-trivia-gym/support.html |
| **Privacy Policy URL** | https://rwnd.github.io/apps/app-trivia-gym/privacy.html |

Local paths:

- [`app-trivia-gym/support.html`](app-trivia-gym/support.html)
- [`app-trivia-gym/privacy.html`](app-trivia-gym/privacy.html)
- Screenshots: [`app-trivia-gym/screenshots/`](app-trivia-gym/screenshots/)

### Enable GitHub Pages

1. Repo **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / folder: `/ (root)`
4. Save, wait 1–2 minutes, then open the URLs above

Paste those two URLs into App Store Connect (Support URL + App Privacy).
